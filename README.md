# Fract-ol 42

#### Factal generator made in c

## Usage

First clone the project and open the `Bonus` directory.

Compile the MinilibX Lib running

```sh
make mlx
```

Then compile the project running

```sh
make all
```

Finally run `./fractol` and enjoy the beauty of fractals :)
