# Fract-ol 42

#### Factal generator made in c

## Usage

First clone the project and compile the MinilibX Library running

```bash
$> make mlx
```

Then compile the project running

```bash
$> make bonus
```

Finally run `./fractol` and enjoy the beauty of fractals :)
